Paint the Town Red
-----

Please vote for a much bigger version to be released on Steam Greenlight: http://southeastgames.com/paintthetownred/greenlight


Controls:

WASD / Arrow = Move
Hold Shift = Run
Mouse = Look Around
Click Left Mouse Button = Stab
Hold Left Mouse Button = Swing
Hold Right Mouse Button = Block
F = Throw held weapon
E = Pickup weapon
G = Drop weapon
Q = Use special attack (when meter is full)
